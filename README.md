# Secure Cloud-Based MLOps Framework for AI-Based Lung Disease Detection

This project is a local prototype for AI-based lung disease detection from chest X-ray images.  
The system will allow a user to upload a chest X-ray image, process it using a trained AI model, and return a prediction result with confidence score.

## Project Structure

- dataset/ - Dataset placeholder
- notebooks/ - Model training notebooks
- model/ - Saved trained model
- app/ - Local web/API application
- aws/ - Cloud deployment notes
- testing/ - API and model testing files
- results/ - Evaluation outputs and graphs
